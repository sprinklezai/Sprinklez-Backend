Hostinger-ready WebApp

This repository now includes a clean static homepage that is ready for Hostinger deployment.

For Hostinger, the entry point is index.html. Upload this file and any supporting assets to your site folder (usually public_html) and the page will load automatically.

Deploy steps:
  1. Upload the repo files to Hostinger via File Manager or FTP.
  2. Make sure index.html is in the site root.
  3. Open your Hostinger domain to view the page.
